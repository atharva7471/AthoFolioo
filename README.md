# 🚀 Atho-Folio — Personal Portfolio Website

A modern and responsive **personal portfolio website** built using **HTML, CSS, JavaScript, and Flask**.  
This website showcases my **projects, skills, resume, achievements, and contact information** in a clean and interactive way.

---

## 🌟 Features

- 🧑‍💻 **About Me Section**  
- 📂 **Projects Showcase** (Movie Recommender, Chatbot, Web Apps, etc.)  
- 📝 **Downloadable Resume (PDF)**  
- 🌙 **Clean and modern UI/UX**  
- 📱 **Fully Responsive for all devices**  
- ⚡ **Fast loading & SEO-friendly**  
- 🌐 **Deployed on Vercel**  

---

## 🛠️ Tech Stack

### **Frontend**
- HTML5

- JavaScript  
- Responsive Design  

### **Backend**

## 🔧 Setup & Run Locally

### 1️⃣ Clone Repository
```bash
git clone https://github.com/atharva7471/atho-folio.git
```

### 2️⃣ Install Dependencies
```bash
pip install -r requirements.txt
```

### 3️⃣ Run the Server
```bash
python app.py
```

Your portfolio will run at:  
👉 **http://127.0.0.1:5000/**
